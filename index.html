List of HTML Tags

Basic Structure Tags

  1. <html>: The root element that wraps the entire HTML document.
  2. <head>: Contains metadata about the document, such as the title and character encoding.
  3. <title>: Specifies the title of the web page, which is displayed in the browser's title bar or tab.
  4. <body>: Encloses the content of the web page that is displayed in the browser window.

Heading and Paragraph Tags

  1. <h1> to <h6>: Heading tags used to define different levels of headings, with <h1> being the highest level.
  2. <p>: Defines a paragraph of text.

Link and Image Tags

  1. <a>: Creates a hyperlink to another web page or a specific location within the same page.
  2. <img>: Inserts an image into the web page.

List Tags

  1. <ul>: Represents an unordered (bullet) list.
  2. <ol>: Represents an ordered (numbered) list.
  3. <li>: Defines a list item within <ul> or <ol>.

Container Tags

  1. <div>: A generic container used to group and style elements.
  2. <span>: Inline container used to apply styles or manipulate specific parts of the text.

Table Tags

  1. <table>: Defines a table with rows (<tr>), table headers (<th>), and table cells (<td>).
  2. <thead>: Groups the header content in a table.
  3. <tbody>: Groups the body content in a table.
  4. <tr>: Defines a row in a table.
  5. <th>: Defines a header cell in a table.
  6. <td>: Defines a data cell in a table.

Form Tags

  1. <form>: Creates a form for user input and interaction.
  2. <input>: An input field that allows users to enter data.
  3. <button>: Represents a clickable button.
  4. <textarea>: A multi-line input field for entering larger text content.
  5. <select>: Creates a dropdown list from which users can select options.
  6. <option>: Defines an option within a <select> dropdown.
  7. <label>: Provides a text label for an input field.
  8. <fieldset>: Groups related form elements.
  9. <legend>: Defines a caption for the <fieldset> element.

Structural Tags

  1. <header>: Represents the introductory content or a group of navigational aids at the top of a page or section.
  2. <footer>: Represents the footer section of a document or a section.
  3. <nav>: Defines a section containing navigation links.
  4. <section>: Represents a standalone section within a document.
  5. <article>: Defines an independent piece of content within a document.
  6. <aside>: Represents content that is tangentially related to the main content, such as sidebars or pull quotes.

Media Tags

  1. <video>: Embeds video content into a web page.
  2. <audio>: Embeds audio content into a web page.
  3. <source>: Specifies media resources for the <video> and <audio> elements.
  4. <iframe>: Embeds external web content within an HTML document.

Other Tags

  1. <br>: Represents a line break.
  2. <hr>: Represents a thematic break or horizontal rule.
  3. <style>: Defines style information for a document.
  4. <script>: Embeds or references external JavaScript code within an HTML document.
